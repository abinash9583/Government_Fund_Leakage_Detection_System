# Government Fund Leakage Detection System

This project is an SAP ABAP analytical solution designed to detect abnormal government fund utilization patterns by comparing allocated budgets with actual spending across departments and sectors.

## Key Features
- Rule-based detection of abnormal fund usage
- Deviation and risk classification logic
- ALV reporting with visual risk indicators
- Government/PSU-oriented financial analytics use case

## Technologies Used
- SAP ABAP
- Open SQL
- Internal Tables (Standard, Sorted)
- ALV Grid
- Modularized ABAP Programming

## Business Use Case
Helps government departments identify potential fund leakage such as early fund exhaustion, overspending, and unusual spending spikes, enabling timely corrective actions.

## Output
An ALV report highlighting departments/sectors with Low, Medium, or High risk based on predefined rules.
